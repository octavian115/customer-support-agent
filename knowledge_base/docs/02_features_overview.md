# TaskFlow Features Overview

## Boards and Lists

Every project in TaskFlow is organized as a board. Boards contain lists (vertical columns), and lists contain task cards. Drag and drop cards between lists to update their status.

You can create up to 20 lists per board on the Free plan, and unlimited lists on Pro and Business plans.

## Tasks

Tasks are the core unit of work in TaskFlow. Each task card supports:

- **Title and Description**: Use markdown in descriptions for rich formatting.
- **Assignees**: Assign one or multiple team members. All assignees receive notifications.
- **Due Dates**: Set a due date and optional due time. Tasks turn yellow 24 hours before the deadline and red when overdue.
- **Labels**: Color-coded tags for categorizing tasks (e.g., "Bug," "Feature," "Urgent"). Create custom labels in project settings.
- **Priority Levels**: Set priority as Low, Medium, High, or Critical. Sort and filter by priority.
- **Checklists**: Add subtask checklists within a card. Track completion percentage on the card face.
- **File Attachments**: Attach files up to 25 MB each. Supported formats include images, PDFs, documents, and spreadsheets. Free plan allows 100 MB total storage per workspace; Pro allows 10 GB; Business allows 100 GB.
- **Comments**: Leave comments with @mentions to notify specific team members. Comments support markdown and file attachments.

## Activity Log

Every task has a full activity log showing who did what and when — assignments, status changes, due date modifications, comments, and file uploads. This log cannot be edited or deleted.

## Views

TaskFlow offers multiple ways to view your work:

- **Board View**: The default Kanban-style layout.
- **List View**: A flat table of all tasks with sortable columns.
- **Calendar View**: Tasks plotted on a calendar by due date. Drag tasks to reschedule.
- **Timeline View** (Pro and Business only): A Gantt-style chart showing task durations and dependencies.

## Search and Filters

Use the global search bar (`Ctrl+K`) to find tasks, projects, or people across your workspace. Filter tasks by assignee, label, priority, due date range, or status. Save frequently used filters for quick access.

## Templates

Save any board as a template. Use templates to standardize workflows across projects. TaskFlow includes built-in templates for Software Development, Marketing Campaigns, HR Onboarding, and Event Planning.
